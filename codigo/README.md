# Código

Esta pasta contém os notebooks principais do projeto.

- `COLAB_01_FASE_EXPLORATORIA.ipynb`: comparação exploratória entre KMeans, DBSCAN e agrupamento hierárquico.
- `COLAB_02_PIPELINE_ARTIGO.ipynb`: pipeline final do artigo, geração do artefato com clusters e avaliação supervisionada.

Os notebooks foram pensados para o Google Colab e usam caminhos do Google Drive. Para execução local, ajuste `DATASET_PATH`, `RESULTADOS_DIR`, `GRAFICOS_DIR` e, no pipeline principal, `ARTEFATO_PATH`.
