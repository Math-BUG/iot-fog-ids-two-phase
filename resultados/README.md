# Resultados

Esta pasta reúne tabelas e sínteses geradas pelos notebooks.

- `exploratorio/`: métricas da comparação entre KMeans, DBSCAN e agrupamento hierárquico.
- `pipeline/`: métricas supervisionadas, matrizes de confusão, pureza de clusters, cobertura e distribuições.
- `comparativo_metricas_exploratorias.md`: resumo interpretativo das melhores configurações observadas na etapa exploratória.

Os arquivos `.csv` foram mantidos separados por etapa para facilitar a leitura e a reprodução das análises.
